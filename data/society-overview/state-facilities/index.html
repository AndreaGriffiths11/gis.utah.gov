Locations of State Facilities

Download Dataset

State Facilities

Data Type: GIS Data Layer
Steward(s): AGRC & Other State Agencies

Abstract:
Society.StateFacilities contains locations for facilities in the state of Utah that are controlled by the state or by entities of the state of Utah. This dataset was created by AGRC with the help of several state agencies and entities. Not all facilities have a geographic location in this dataset.

Last Update: 2011
